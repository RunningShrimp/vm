## 🚀 部署建议

### 环境要求

**最低要求**:
- Rust 1.70+
- 64位操作系统
- 4GB+ RAM
- 硬件虚拟化支持(可选)

**推荐配置**:
- Rust 1.75+
- 8GB+ RAM
- SSD存储
- 硬件虚拟化(KVM/HVF/WHPX)
- GPU(可选,CUDA 11.0+或ROCm)

### 部署步骤

1. **编译项目**
   ```bash
   cargo build --release
   ```

2. **运行测试**
   ```bash
   cargo test --workspace
   ```

3. **性能基准**
   ```bash
   cargo bench --workspace
   ```

4. **启动VM**
   ```bash
   cargo run --release --bin vm-desktop
   ```

### 配置选项

**启用JIT**:
```bash
export VM_ENABLE_JIT=1
```

**选择加速后端**:
```bash
export VM_ACCEL=kvm  # Linux
export VM_ACCEL=hvf  # macOS
export VM_ACCEL=whpx # Windows
```

**启用GPU**:
```bash
export VM_GPU=cuda  # NVIDIA
export VM_GPU=rocm  # AMD
```
